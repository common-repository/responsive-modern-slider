== Description ==

Plugin slider with advanced features <strong> four types of sliders within the plugin</strong>: <a href="https://garmoniagroup.ru/digitalshop/wordpress-plagin-slayder/">https://garmoniagroup.ru/digitalshop/wordpress-plagin-slayder/</a>
Плагин слайдера с расширенными возможностями <strong> четыре разновидности слайдеров в одном плагине</strong>: <a href="https://garmoniagroup.ru/digitalshop/wordpress-plagin-slayder/">https://garmoniagroup.ru/digitalshop/wordpress-plagin-slayder/</a>
== Installation ==


1. Unzip `responsive-modern-slider.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
